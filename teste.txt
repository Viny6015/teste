git init -> iniciar o repositorio
git status -> mostra o status do repositório
git add -A -> adicionar todos os arquivos ao controle de versão
git commit -m "comentario" -> confirmar o arquivo

git pull -> trazer do github

git remote add origin -> associar o repositorio local com um remoto
